What Does It Do?

This library is an add-on for SuperScript. It is strongly recommended to read the documentation for the core project before proceeding with this library. If you haven't yet read the core documentation then it is recommended that you start by reading about the core concepts.

SuperScript.Container.Mvc contains the core code used by higher-level, more functional SuperScript projects which offer a Razor view-based block for enclosing server-side entities to be processed by SuperScript. 

	public abstract class BaseContainer : IDisposable
	{
	    protected BaseContainer(HtmlHelper helper, string emitterKey, int? insertAt)
	    public abstract void Dispose()
	}

The above abstract class contains the base functionality for retrieving the contents of a derived code block, allowing the derived implementation to move the contents into an implementation of DeclarationBase.
